# Электронный дневник тренера (FitCoach) — веб-версия

Веб-приложение для тренера и клиентов: учёт тренировок, упражнений и подходов
(по модели **LiftLog**), замеры тела, автоматическая статистика и графики,
**система уровней и опыта (XP)**, **рейтинг**, уведомления.

## Что нового (v2.0)

- **🏆 Система уровней и XP** — за каждую тренировку начисляется опыт.
  Чем больше рабочий вес, объём (вес × повторы × подходы), количество подходов
  и повторений — тем больше XP. За побитый личный рекорд — бонус. Набрав XP,
  пользователь повышает уровень и получает звание (Новичок → Легенда).
- **🥇 Рейтинг (лидерборд)** — сортировка клиентов по опыту с местами,
  медалями за топ-3 и подсветкой своего положения.
- **📱 Полная адаптивность** — телефон, планшет, компьютер (модальные окна
  снизу на мобиле, выезжающее меню, крупные кнопки, поддержка `prefers-reduced-motion`).
- **🔒 Безопасность** — надёжность пароля, валидация ввода, JWT-авторизация,
  контроль доступа по ролям, security-заголовки, параметризованные запросы (SQLAlchemy).
- **✅ Тесты** — pytest с позитивными и негативными проверками.
- **▶️ Локальный запуск в один клик** — `start.bat` / `start.ps1`.

## Быстрый запуск (Windows)

1. Двойной клик по **`start.bat`**.
   Скрипт сам: создаст виртуальное окружение, установит зависимости,
   инициализирует базу (SQLite — без установки PostgreSQL), запустит сервер
   и откроет браузер на `http://localhost:8000`.

2. Готово. Откроется лендинг → кнопка «Войти».

> Требуется установленный **Python 3.10+** (с галочкой *Add Python to PATH*).
> Скачать: https://www.python.org/downloads/

### Запуск вручную

```bash
cd python
python -m venv .venv
# Windows:
.venv\Scripts\activate
# Linux/macOS:
source .venv/bin/activate

pip install -r requirements.txt
python init_db.py
python -m uvicorn main:app --reload --port 8000
```

## Тесты

```bash
# Двойной клик run_tests.bat, либо вручную:
cd python
pytest ../tests -v
```

Тесты используют изолированную in-memory SQLite БД для каждого теста.
Покрытие: аутентификация, клиенты, тренировки + начисление XP, геймификация,
рейтинг, замеры — позитивные (happy path) и негативные (ошибки 401/403/404/422/409).

## База данных

По умолчанию используется **SQLite** (`python/fitness.db`) — запуск без настройки.
Для **PostgreSQL** задайте переменную окружения:

```bash
# Windows
set DATABASE_URL=postgresql+psycopg2://postgres:ПАРОЛЬ@localhost:5432/fitness_trainer
```

Затем `python init_db.py` создаст таблицы и демо-данные.

## Как начисляется опыт (XP)

| Источник                        | XP                                |
|---------------------------------|-----------------------------------|
| Завершённая тренировка          | +50 (база)                        |
| Объём (вес × повторы × подходы) | +0.25 за кг (макс. +500)          |
| Каждый подход                   | +5                                |
| Каждое повторение               | +1 (макс. +150)                   |
| Личный рекорд по упражнению     | +100                              |

Уровень растёт нелинейно: `200 × уровень^1.4` XP для перехода на следующий.

## Безопасность

- Пароли хешируются (bcrypt), проверяется надёжность (≥8 символов, буквы + цифры).
- Авторизация по JWT с ролями (trainer / client) и контролем доступа.
- Валидация всех входных данных (Pydantic).
- SQL-инъекции исключены (параметризованные запросы SQLAlchemy).
- Security-заголовки ответов.
- Инфраструктура rate-limiting (slowapi) — доступна для подключения.
